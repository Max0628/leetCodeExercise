                - 1. User can start 1 on 1 chat with it's friend by clicking the button.

tech stack
    - Frontend: React Native for cross-platform mobile app development
    - Backend: Java Spring boot or Python FastAPI for building RESTful APIs
    - Database:
        - RDBMS: PostgreSQL for relational data (user profiles, message metadata)
        - NoSQL: MongoDB or Cassandra for storing message history and media metadata
        - KV Store: MongoDB for fast access to user sessions and presence information
        - In-Memory Data Store: Redis for caching and real-time presence tracking
    - Messaging Queue: Apache Kafka for handling message delivery and notifications
    - Real-time Communication: WebSocket (Socket.io) for real-time messaging
    - Authentication: JWT (JSON Web Tokens) for secure user authentication
    - Media Storage: AWS S3 for storing media files
    - Push Notifications: Firebase Cloud Messaging (FCM) for sending push notifications
    - Deployment: Docker and Kubernetes for containerization and orchestration
    - Observability:
        - Metrics: Prometheus for monitoring application performance
        - Logging: ELK Stack (Elasticsearch, Logstash, Kibana) for centralized
        - Tracing: Jaeger for distributed tracing of requests
    - CI/CD: Jenkins or GitHub Actions for continuous integration and deployment
    - Load Balancing: NGINX or AWS ELB for distributing incoming traffic
    - API Gateway: Kong or AWS API Gateway for managing and routing API requests
    - Security: TLS/SSL for secure data transmission, OAuth 2.0 for authorization
